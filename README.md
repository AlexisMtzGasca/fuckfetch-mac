# FuckFetch for MacOS

A simple fetch script for MacOS

For Install, just:

git clone https://github.com/AlexisMtzGasca/fuckfetch-mac.git && cp -v ./fuckfetch-mac/fuckfetch /usr/local/bin && sudo chmod 777 /usr/local/bin/fuckfetch && fuckfetch
